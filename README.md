# CoHost Management — Intelligent Property Management

> **Live Site:** [https://jordan-millennia.github.io/CoHost-Management-Website](https://jordan-millennia.github.io/CoHost-Management-Website)

National done-for-you property management for Short-Term, Mid-Term, Long-Term, and PadSplit Co-Living rentals. Tech-forward operations, transparent reporting, maximum returns.

## Local Setup

```bash
# Clone the repository
git clone https://github.com/Jordan-Millennia/CoHost-Management-Website.git
cd CoHost-Management-Website

# Open directly in browser — no build step required
open index.html
```

No dependencies to install. The site uses vanilla HTML/CSS/JS with CDN-loaded libraries (Three.js, GSAP, Chart.js).
